## Basic Counter Example

Just write your Laminar code directly in the markdown with `scala preview`:

```scala preview
val countVar = Var(0)

div(
  display.flex,
  alignItems.center,
  justifyContent.center,
  gap := "10px",
  button(
    "-",
    onClick --> Observer { _ =>
      countVar.update(_ - 1)
    }
  ),
  span(
    text <-- countVar.signal.map(_.toString)
  ),
  button(
    "+",
    onClick --> Observer { _ =>
      countVar.update(_ + 1)
    }
  )
)
```
