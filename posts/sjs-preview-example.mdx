## With State (Counter Example)

<Playground preset="sjs">

```scala preview
val countVar = Var(0)
div(
  display.flex,
  alignItems.center,
  gap := "10px",
  button(
    "-",
    onClick --> Observer { _ =>
      countVar.update(_ - 1)
    }
  ),
  span(
    child.text <-- countVar.signal.map(_.toString)
  ),
  button(
    "+",
    onClick --> Observer { _ =>
      countVar.update(_ + 1)
    }
  )
)
```

</Playground>

Note: This is the new automatic approach. No manual mill setup needed!
